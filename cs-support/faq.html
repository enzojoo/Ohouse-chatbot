01

주문·결제

Q어떤 결제 수단을 이용할 수 있나요?▾

오늘의집은 다음 결제 수단을 제공합니다.

오늘의집페이(계좌 간편결제) / 카드 / 무통장입금 / 퀵계좌이체 / 카카오페이 / 토스페이 / 네이버페이 / 페이코 / 휴대폰 결제

Q비회원으로 주문할 수 있나요?▾

iOS 앱과 PC에서는 비회원 구매가 가능하지만, 안드로이드 환경에서는 회원가입 후 이용 가능합니다.

비회원 주문 시 쿠폰·포인트 적용은 불가하며, 주문 취소/교환/반품은 고객센터를 통해서만 처리됩니다.

Q신용카드 무이자 할부가 되나요?▾

각 카드사 정책에 따라 상이하며, 카드사를 통해 확인 가능합니다. 무이자 혜택은 결제 진행 시 확인할 수도 있습니다.

Q무통장입금은 언제까지 입금해야 하나요?▾

주문 완료일 +3일 이내 입금해 주셔야 합니다. 단, 원하는날배송 상품의 희망배송일이 입금 기한보다 빠른 경우 희망배송일과 물류 배송 마감 시간에 맞춰 입금 기한이 변경됩니다.

입금 내역은 입금 후 24시간 이내 자동 확인됩니다. 24시간이 지나도 확인이 되지 않는 경우 입금 금액과 계좌가 정확한지 확인 후 고객센터로 문의해 주세요.

Q가상계좌 입금자명을 다르게 적었어요.▾

입금자명이 다르더라도 가상계좌번호가 일치하면 정상적으로 입금 확인됩니다.

Q결제 후 결제 방법을 변경할 수 있나요?▾

결제 완료 후 결제 정보 변경은 불가능합니다. 단, 입금대기 및 결제완료 단계에서는 취소 후 재주문을 통해 변경 가능합니다.

Q다른 사람의 카드로 결제할 수 있나요?▾

카드 소유주의 동의가 있다면 가능합니다. 단, 카드 결제 시 [다른결제 > 일반 결제]로 진행해야 하며, 최초 시도 시 카드 소유주의 휴대폰/공동인증서 인증이 필요할 수 있습니다.

Q결제 시 에러 메시지가 나옵니다.▾

[마이페이지 > 나의쇼핑 > 고객센터]를 통해 문의 부탁드립니다.

Q주문 후 배송지나 수령인 정보를 변경할 수 있나요?▾
  • 결제완료 단계: 배송지·수령인 정보를 [마이페이지 > 나의쇼핑 > 주문상세]에서 직접 변경 가능
  • 배송준비중 단계 이후: 직접 변경 불가. 판매자 확인 필요
Q한 번에 몇 개까지 주문할 수 있나요?▾

선택하신 옵션 기준으로 한 번에 30개까지만 주문 가능합니다.

Q해외직구 상품 주문 시 개인통관고유부호가 오류 납니다.▾

개인통관고유부호는 배송지 수령인의 [성함, 연락처, 개인통관고유부호] 3가지가 모두 일치해야 정상 진행됩니다.

  • 주문자가 아닌 수령인 정보 기준입니다.
  • 영문 P + 숫자 12자리 (총 13자) 정확히 입력해 주세요.
  • 시작·끝 공백·특수문자가 없도록 확인 부탁드립니다.

정보가 일치하는데도 오류가 발생하면 관세청 유니패스(unipass.customs.go.kr)에서 사용 상태를 확인해 주세요.

02

배송

Q배송 유형은 어떤 것이 있나요?▾
  • 일반 택배 (CJ대한통운, 롯데택배, 우체국택배 등): 운송장 부여 후 배송추적 가능
  • 화물 택배 (경동택배, 대신택배 등): 일반 택배로 어려운 대형 물품
  • 업체 직접 배송: 가구류 등, 배송기사 해피콜 후 배송. 배송 흐름 조회 불가
  • 해외배송 (해외직구): 개인통관고유부호 필수
  • 원하는날 도착: 오늘의집이 직접 가구 전문 배송·설치
Q배송비는 얼마인가요?▾

상품 및 판매자별로 배송비 정책이 상이합니다. 각 상품 상세 페이지에서 정확한 배송비를 확인하실 수 있습니다. 특히 가구 상품의 경우 배송 지역이나 엘리베이터 유무, 사다리차 사용 등 배송 환경에 따라 배송비가 달라질 수 있으니 상세 페이지의 배송비 정보를 정확하게 확인해 주세요.

  • 지역별 차등 배송비: 도서산간 등 일부 지역은 추가 배송비 발생
  • 묶음 배송: 같은 파트너사 상품이더라도 물류 사정에 따라 묶음 배송이 어려울 수 있습니다.
Q배송은 얼마나 걸리나요?▾

배송 유형에 따라 상이하며, 상품 상세 페이지에서 정확한 배송 일정을 확인하실 수 있습니다.

  • 일반/화물 택배: 결제 후 영업일 기준 출고
  • 오늘출발 상품: 평일 마감 시간 이전 주문 시 당일 출고
  • 업체 직접 배송: 배송 지역에 따라 일정 상이
  • 원하는날 도착: 오전 11시 이전 결제 시 내일 도착 (일부 지역 제외, 일요일·공휴일 제외). 최대 4개월(120일) 이내 원하는 날짜 지정 가능
Q배송 시작 알림을 받았는데 배송 추적이 안 됩니다.▾

송장번호 등록 후 1영업일 이내 또는 실제 상품 배송이 진행되면서 추적이 가능합니다. 배송처에 따라 운송장만 먼저 등록되는 경우가 있습니다.

Q배송완료로 표시되는데 상품을 받지 못했습니다.▾

경비실 또는 무인택배함을 먼저 확인해 주세요. 별도의 위탁 장소가 없는 경우 배송기사가 임의 장소에 보관하셨을 수 있으니 운송장의 기사 연락처로 먼저 문의 부탁드립니다. 분실 의심 시 고객센터로 문의해 주시면 판매자·택배사를 통해 확인을 도와드립니다.

Q원하는 날짜에 배송받을 수 있나요?▾

'원하는날 도착'·'빠른가전배송' 뱃지가 있는 상품은 주문 시 직접 배송일 지정이 가능합니다. 그 외 상품은 상품 페이지 또는 파트너사 정책에 따라 가능 여부가 다릅니다.

Q해외배송이 가능한가요?▾

오늘의집 상품은 국내 배송만 가능합니다.

Q배송기사 연락처를 알 수 있나요?▾

운송장 조회가 되는 상품은 송장에서 배송기사 연락처를 확인할 수 있습니다. 업체 직접 배송 상품은 배송 전 또는 배송 당일 기사님의 해피콜이 진행되며, 별도 연락처는 파트너사로 문의해 주세요.

03

취소·교환·반품

Q배송 상태별로 취소 가능 여부가 다른가요?▾

결제완료 상태일 경우 즉시 취소가 가능하며, 배송준비중 상태의 경우 판매자 확인 후 취소 처리가 진행됩니다. 판매자 확인 결과 이미 상품이 출고된 상태라면 수령 후 반품으로 진행됩니다.

Q교환·반품 신청 기간이 정해져 있나요?▾

상품 수령일로부터 7일 이내 (구매확정 이전)에 교환·반품 신청이 가능합니다. 단, 아래의 경우 교환·반품이 제한됩니다.

  • 고객의 책임 사유로 상품이 멸실·훼손된 경우
  • 개봉·포장 훼손으로 상품 가치가 현저히 상실된 경우
  • 시간 경과로 재판매가 어려울 정도로 상품 가치가 저하된 경우
  • 주문 후 제작에 들어간 주문 제작 상품
  • 직접 조립(DIY) 상품을 조립한 경우
  • 매트리스 등 비닐 포장 제거 시 위생 문제로 재판매 불가한 상품
Q교환·반품은 어떻게 신청하나요?▾
  1. 마이페이지 → 나의쇼핑 → 주문배송목록 → 해당 주문 [교환요청] 또는 [반품요청]
  2. 사유 입력 (구매자 귀책 / 판매자 귀책)
  3. 판매자 귀책(불량·오배송)인 경우 사진 첨부 권장
  4. 회수 방법 선택 ('직접 수거해주세요' 또는 '이미 택배로 보냈어요')
Q교환·반품 배송비는 어떻게 되나요?▾
  • 단순변심 (고객 귀책): 반품·교환 배송비를 고객이 부담
    • 무료배송 상품 전체 반품 시: 최초배송비 + 반품배송비 (왕복) 발생
    • 부분 반품 시: 반품 회수비용만 발생
  • 불량·오배송 (판매자 귀책): 배송비 판매자 부담

Q사은품도 함께 반품해야 하나요?▾

사은품도 함께 동봉해 주셔야 반품 처리가 완료됩니다.

Q일부 수량만 반품·교환할 수 있나요?▾

동일 옵션 상품을 여러 개 주문하신 경우, 부분 취소·반품·교환은 마이페이지에서 직접 접수가 불가합니다. 고객센터를 통해 처리 가능하며, 단순변심의 경우 전체 반품 후 필요 수량 재주문으로 안내될 수 있습니다.

Q'원하는날 도착' 상품도 반품할 수 있나요?▾
  • 조립·설치·포장 개봉 후 사용으로 재판매가 어려운 경우, 전자상거래법에 따라 교환·반품이 제한될 수 있습니다.
  • 제품이 표시·광고 내용과 다른 경우: 공급일로부터 3개월 이내 또는 사실을 안 날로부터 30일 이내 무상 교환·반품 가능
  • 제품 하자: 상품 자체의 하자로 인해 사용이 불가능한 경우 사유 확인 후 반품 가능
Q반품 접수했는데 그냥 사용하고 싶어요. 철회가 되나요?▾

상품이 회수되기 전인 '반품요청·교환요청' 상태라면 직접 철회가 가능합니다. 단, 택배사로 회수 데이터가 이관된 이후에는 철회가 불가하므로 회수 기사님의 연락에 "수거할 상품이 없다"고 안내 부탁드립니다.

04

영수증·증빙

Q세금계산서를 받을 수 있나요?▾

오늘의집은 세금계산서를 발급하지 않습니다. 동일한 효력을 가진 '지출증빙용' 현금영수증을 신청해 주세요. 현금영수증을 원치 않을 경우 법인카드 결제를 권장 드립니다.

Q현금영수증을 발급받으려면 어떻게 해야 하나요?▾

무통장입금 결제 시 주문서에서 직접 입력해야 합니다.

  • 소득공제용: 휴대폰 번호 또는 주민등록번호
  • 지출증빙용: 사업자번호

별도로 입력하지 않으셔도 10만원 이상 결제 시 국세청 지정 코드(010-***-1234)로 자진발급됩니다.

발급 정보 변경

  • 자진발급의 경우: 국세청 홈택스에서 '자진발급분 소비자등록'으로 직접 등록
  • 정보 오입력으로 발행이 실패한 경우: 홈택스에서 직접 수정
  • 그 외 정정이 어려운 경우: 고객센터로 문의
Q결제 영수증은 어디서 확인할 수 있나요?▾
마이페이지 → 나의쇼핑 → 주문배송내역조회 → 주문상세 → 결제정보 옆 [영수증] 클릭

네이버페이/토스페이 결제 영수증은 해당 결제사 서비스에서 확인 가능합니다.

Q견적서를 발급받을 수 있나요?▾

가능합니다. 고객센터로 아래 정보와 함께 요청해 주세요: 상품명·옵션명·수량·상품 URL·요청사항·이메일 주소·발행 사유. 견적서는 이메일로 발송되며, 유효기간은 1개월이나 그 사이 가격이 변동될 수 있습니다.

Q거래명세서를 받을 수 있나요?▾

주문 상태가 [배송완료] 또는 [구매확정]인 경우에만 발급 가능합니다. 주문번호 전체 발급만 가능하며, 일부 옵션만 발행은 어렵습니다.

05

회원·로그인

Q회원가입은 어떻게 하나요?▾
  • PC/웹: 오늘의집 → 회원가입 → 이메일 일반가입 또는 간편가입(카카오톡/네이버/페이스북)
  • 모바일/앱: 마이페이지 → 회원가입 → 이메일 일반가입 또는 간편가입(카카오톡/네이버/페이스북/애플)
    • iOS에서는 애플 로그인 가능
    • Android는 회원가입 후에만 앱 이용 가능

이메일 가입 시 입력한 이메일로 인증이 필요하며, '개인정보 마케팅 활용 동의' 및 '이벤트/쿠폰/특가 알림' 수신은 선택 사항입니다.

Q회원정보(이메일·닉네임)는 어떻게 변경하나요?▾
  • 이메일 변경: 마이페이지 → 설정 → 회원정보수정 → 새 이메일 입력 → 이메일 인증
    • 일반(이메일) 가입 계정에서만 직접 변경 가능. 간편가입 계정은 고객센터로 문의해 주세요.
  • 닉네임 변경: 마이페이지 → 설정 → 회원정보수정 → 새 닉네임 입력
    • 2~20자, 한글·영어·숫자·한문 가능
Q비밀번호를 잊어버렸어요.▾
로그인 화면 → '비밀번호 재설정' → 가입 이메일 입력 → 이메일로 받은 인증코드 입력 → 새 비밀번호 설정

일반(이메일) 가입 계정에서만 가능합니다. 간편가입 계정은 가입하신 SNS에서 비밀번호 확인이 필요합니다.

Q간편 로그인(카카오/네이버/페이스북/애플)으로 로그인이 안 됩니다.▾

상황별로 안내드립니다.

  • 휴대폰 번호 변경 / SNS 계정 탈퇴·재가입: 새 정보 반영을 위해 고객센터(1670-0876)로 문의해 주세요. 가입 정보 매칭 또는 이메일 계정으로 전환을 도와드립니다.
  • 카카오톡 간편가입 후 '이미 가입된 이메일' 알림: 카카오톡 [설정 → 카카오계정 → 연결된 서비스관리 → 외부서비스 → 오늘의집]에서 회원번호 확인 후 고객센터로 알려주세요.
  • 애플 로그인: 웹사이트에서는 애플 로그인이 불가하며, 모바일 앱에서만 가능합니다.
  • 간편가입 후 SNS 계정을 더 이상 사용하지 않는 경우: 고객센터를 통해 이메일 계정으로 전환이 가능합니다.
Q회원탈퇴는 어떻게 하나요?▾
  • PC: 마이페이지 → 설정 → 회원정보수정 → 탈퇴하기
  • 모바일 앱: 마이페이지 → 설정 → 내 정보 수정 → 탈퇴하기
  • 전문가 회원: pro@bucketplace.net으로 문의

탈퇴 시 안내사항

  • 오늘의집 포인트·쿠폰은 모두 소멸되며 환불되지 않습니다.
  • 진행 중인 주문건은 비회원 주문으로 전환되어 [비회원 주문조회]로 확인 가능합니다.
  • 작성하신 게시물·댓글은 자동 삭제되지 않으므로, 삭제를 원하시는 경우 탈퇴 전 직접 삭제 부탁드립니다.
  • 관련 법령에 따라 거래 기록은 5년, 분쟁 처리 기록은 3년간 보관됩니다.
  • 재가입은 즉시 가능하나 기존 포인트·구매 정보는 연동되지 않습니다.
Q회원등급은 어떻게 결정되나요?▾
  • WELCOME: 회원가입 시 즉시 적용. 적립률은 매월 상이 (자세한 사항은 이벤트 페이지 확인)
  • VIP: 최근 3개월 동안 구매확정 6회 이상 + 실 결제금액 50만원 이상. 실 결제금액의 3% 적립
    • 매주 토요일 자정 집계, 일요일 오전 10시 등급 변경

확인 경로: 마이페이지 → 나의쇼핑 → 구매등급

07

오늘의집페이 (간편결제)

Q오늘의집페이는 무엇인가요?▾

오늘의집페이는 오늘의집 자체 간편결제 서비스로, 계좌 간편결제 방식으로 운영되며 보유한 예금 계좌를 연결해 공인인증서나 카드 번호 등록 없이 간단히 결제할 수 있습니다.

  • 14세 미만 가입 불가, 법인폰 사용 불가
  • 본인 명의 계좌만 등록 가능 (최대 10개)
Q오늘의집페이 가입은 어떻게 하나요?▾

오늘의집 계정이 있더라도 오늘의집페이는 별도 회원가입이 필요합니다.

  1. 필수 약관 동의
  2. 본인 인증 / 휴대전화번호 인증
  3. 결제 비밀번호 등록
  4. 계좌 등록 (은행 선택 → 약관 동의 → ARS 인증)

ARS 안내 멘트가 종료된 후 인증번호 2자리를 입력해야 합니다. 휴대폰 명의와 계좌 명의가 동일해야 합니다.

Q오늘의집페이 결제 한도는 얼마인가요?▾
  • 1회 1,000만원 / 1일 누적 2,000만원 / 월 누적 2,000만원
Q오늘의집페이 결제 혜택은 무엇인가요?▾
  • 일반 결제 적립: 결제 금액의 2%를 오늘의집 포인트로 적립 (구매확정 시점에 지급, 유효기간 365일)
  • 회원등급별 기본 적립과 별도 지급
Q오늘의집페이에서 현금영수증을 발급받을 수 있나요?▾

가능합니다. 마이페이지 → 쇼핑 → 결제수단 → 현금영수증 설정에서 용도와 정보 입력이 가능합니다.

Q오늘의집페이를 해지하고 싶어요.▾
마이페이지 → 쇼핑 → 결제수단 → 서비스 해지

서비스를 해지하더라도 오늘의집 계정은 유지됩니다.

Q등록이 불가한 계좌가 있나요?▾

다음 계좌는 등록이 불가합니다.

  • 본인 명의가 아닌 계좌
  • 법인 사업자 계좌
  • 휴대폰 번호 등 평생 계좌번호
  • 가상 계좌번호
  • 펀드/적금/정기예금/외화 등의 계좌
Q오늘의집페이 환불은 언제 되나요?▾

결제 시 사용한 계좌로 1~2영업일 이내 환불됩니다. 계좌 상태에 따라 자동 환불이 어려운 경우 다른 계좌로 수기 환불이 진행됩니다.

Q작성한 리뷰가 블라인드 처리되었어요. 기준이 무엇인가요?▾

오늘의집 서비스 이용 약관 제31조에 따라 다음 금지행위에 해당하는 리뷰는 통보 없이 삭제 또는 블라인드 처리될 수 있습니다.

  1. 특정 업체·개인의 영리적 목적을 위한 리뷰 (판매자 본인·이해관계인 게시, 경쟁사 비방, 대가성 리뷰의 미고지 등)
  2. 허위·과장된 내용 또는 직접 작성하지 않은 내용·사진 (상품 페이지 캡쳐, 타·유사상품 리뷰 등)
  3. 타인의 권리 침해 (개인정보·사생활 노출, 지식재산권·명예·신용 등)
  4. 욕설·폭언·비방, 음란물·청소년 유해 매체, 범죄 조장 등 반사회적 내용
  5. 악성코드·피싱·사기성 사이트 링크 등 부적절 내용
  6. 기타 관련 법령·이용약관·운영정책에 위배되는 경우

자세한 사항은 고객센터로 문의 부탁드립니다.
